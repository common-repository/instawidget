=== InstaWidget===
Tags:Instagram,widget
Requires at least: 2.0
Tested up to: 3.4
Stable tag: 1.0

A simple widget that shows recent Instagram Images.

== Description ==

A very simple widget that displays a recent image from instagram, along with a link to that instagram account. 
 

== Installation ==

1. Install the Plugin
2. Login to Wordpress Admin and activate the plugin.
3. Sign up for Instagram API key at: http://instagram.com/developer/
4. Create an Application to gain a Key and Secret
5. On Widget page enter your Key and Secret
